Requirements
- [Material Design Iconic Font](https://zavoloklom.github.io/material-design-iconic-font/) for icons
- [Iosevka Nerd Font Mono](https://www.nerdfonts.com/) or any nerd font for icons
- `xbacklight` for brightness
- `pamixer` for volume
- `nmcli` for network
- An [OpenWeatherMap](https://openweathermap.org/) api key in `daemons/weatherkey.lua` (also change your city id at `daemons/weather.lua`)
- [Bling](https://github.com/BlingCorp/bling)
- [Rubato](https://github.com/andOrlando/rubato) for animations
